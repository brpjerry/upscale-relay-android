**This build is signed. You must uninstall any earlier version before installing it.**

Every previous release published an unsigned APK, which Android refuses to
install — it had to be signed by hand first. This release is signed during the
build, so the attached APK installs directly.

Because the signature changes, Android treats this as a different app and will
refuse to install it over an existing copy. Uninstall the old build first.
Playback history and settings are stored with the app and will not survive that
uninstall.

Future releases are signed with the same key and upgrade normally; this
one-time break applies only to moving off the older unsigned builds.

There are no application changes in this release. It is identical to v0.14.0
apart from being signed.
